# gsplat-windows-builds

Unofficial [gsplat](https://github.com/nerfstudio-project/gsplat) precompiled Windows binary wheel builder via [GitHub Actions](.github/workflows/build-windows-wheels.yml).

## Automatic Installation

Run `install_gsplat_windows.py` in the environment (venv) where you want to use gsplat.
```
python install_gsplat_windows.py
```

## Manual Installation

Download the appropriate file from the [release page](https://github.com/nagadomi/gsplat-windows-builds/releases) and install it.
